# **Rutas de conocimiento - Analista de datos :artist:**

¡Bienvenidos! ¿Alguna vez has sentido que tus datos no están siendo aprovechados al máximo? ¿Te gustaría tener las herramientas necesarias para extraer información valiosa de ellos y utilizarla para tomar decisiones informadas en tu empresa o proyecto personal? Entonces este curso es para ti.

El análisis de datos es una habilidad esencial en el mundo actual, donde la cantidad de información que se genera cada día es abrumadora. El objetivo de este curso es brindarte los conocimientos y habilidades necesarias para analizar y comprender los datos que tienes a tu disposición, y así poder utilizarlos de manera efectiva para mejorar tus resultados.

No se requiere experiencia previa en análisis de datos. Lo importante es tener la actitud de querer aprender y estar dispuesto a poner en práctica los conocimientos adquiridos. ¡Únete a nosotros en este emocionante viaje hacia el mundo del análisis de datos y descubre el potencial de tus datos!

## Analista de datos de Power BI - Microsoft :page_facing_up:
- **Descripción:** El analista de datos de Power BI ofrece información procesable al aprovechar los datos disponibles y aplicar la experiencia en el dominio. El analista de datos de Power BI colabora con las partes interesadas clave en las verticales para identificar los requisitos comerciales, limpia y transforma los datos, y luego diseña y crea modelos de datos mediante el uso de Power BI. El analista de datos de Power BI proporciona un valor comercial significativo a través de visualizaciones de datos fáciles de comprender, permite que otros realicen análisis de autoservicio e implementa y configura soluciones para el consumo.
